# Voting App 🗳️

A distributed voting application built with Docker, 
Python, Node.js, .NET and Redis.

## Tech Stack
- **Vote** — Python / Flask
- **Worker** — .NET 7 / C#
- **Result** — Node.js / Express / Socket.io
- **Queue** — Redis
- **Database** — PostgreSQL
- **Infra** — Docker Compose / Kubernetes

## Run locally
docker compose up --build

- Vote: http://localhost:8080
- Results: http://localhost:8081
